# Ray Tracing PBR

- 团队名：追光小队
- 项目名：基于PBR和SDF的光线追踪渲染器

## 项目介绍

- 大概计划项目的过程：
- - 先写 GLSL 实现原型，使用 Godot 编辑（因为我已经写好了摄像机控制和降噪管线）
- - 使用 Taichi 重构基础代码，并加上更多优化和功能
- - 预计完成时间两周半

- 项目内容：
- - 支持 PBR 、 Ray Marching 的光线追踪渲染器，支持 SDF 地图
- - 地编、后处理、加几个简单动画（程序性动画）

- 硬件说明：
  - AMD Ryzen 7 6800H with Radeon(TM) Graphics
